Patterns and Template Library for Block and FSE Editor

== Description ==

TemplateGalaxy is a powerful WordPress plugin designed to enhance your block Full Site Editing (FSE) experience by providing an extensive library of patterns and templates. Whether you’re a developer, designer, or content creator, TemplateGalaxy simplifies the process of designing and organizing your website using the block FSE editor.

= Available Patterns and Templates =
<ul>
<li>Header Layouts</li>
<li>Footer Layouts</li>
<li>Homepage Template</li>
<li>Featured Banner/Hero Section</li>
<li>About us Section</li>
<li> Service block Section</li>
<li>Call To Action Section</li>
<li>Testimonial Section</li>
</ul>

== Installation ==

= DIRECT FROM THE DASHBOARD =
1. Go to Dashboard ->Plugins -> Add New
2. Search for "TemplateGalaxy"
3.Now Install and Activate the Plugin

= DOWNLOAD & UPLOAD MANUALLY =
1. Go to https://wordpress.org/plugins/ and Search for "TemplateGalaxy" and download file.
2. Extract File and placed it to the “/wp-content/plugins/” directory.
3. Activate the plugin through the “Plugins” menu in WordPress.


== Frequently Asked Questions ==

= Is TemplateGalaxy is a free plugin? =
Yes, it's free.

Yes, you can use this plugin with any themes that support or comes with block full site editing features.
